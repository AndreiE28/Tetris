# Tetris
A well-known puzzle game where falling blocks must be arranged to form complete horizontal lines, which then disappear. The game speeds up as the player progresses.
# Controls
Q - quit
<br>E - hold
<br>Space - hard drop
<br>W or Up Arrow - rotate clockwise
<br>S or Down Arrow - soft drop
<br>A or Left Arrow - move left
<br>D or Right Arrow - move right
